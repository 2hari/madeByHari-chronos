### Project overview

A fully fluid responsive Figma to React UI website with scroll animations. The animation uses the awesome "aos" library, which makes it super easy to implement scroll reveal animations.

[Compare with Figma design](https://www.figma.com/file/A18UyNjMwCvaOh0mBwcVES/chronos?type=design&node-id=0%3A1&t=iRCNdNiPNGHhuWMG-1)
